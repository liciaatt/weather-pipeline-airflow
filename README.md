# Pipeline Project — Data Engineering

## Description
Pipeline de données complet construit avec :
- Apache Airflow — orchestration
- PostgreSQL — stockage
- Docker — containerisation
- Python — extraction et transformation
- CI/CD GitHub Actions — déploiement automatique

## Architecture
API publique → Python → PostgreSQL → Airflow orchestration

## Comment lancer le projet
```bash
docker-compose up
```
